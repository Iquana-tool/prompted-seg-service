# Prompted Segmentation
This repo contains the API for the prompted segmentation service of the CORAL tag application.

## Running the service
### 1. Using the [uv package manager](https://docs.astral.sh/uv/)
1. Install uv following their installation guide
2. Run the following:
```shell
uv sync
uv run fastapi run main.py --port [YOUR PORT]
```
